# Bill of Materials (BOM) Template

> Template untuk daftar HPP (Harga Pokok Penjualan) produk NGEMILOH franchise.
> Owner mengisi data produk sesuai dengan harga supplier aktual.

## Cara Penggunaan

1. Copy template ini ke file baru (misalnya `BOM_2026.csv`)
2. Isi kolom `hpp` dengan harga pokok dari supplier
3. Update secara berkala setiap ada perubahan harga supplier
4. HPP digunakan untuk perhitungan profit margin di laporan keuangan
